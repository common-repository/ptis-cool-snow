== Description ==

This is the best snow script I ever seen. It based on XSnow written by Nikolaus Klepp (C)2002 GPL.

So I ported this to wordpress and added some options.
This plugin can automaticly detect winter and runs script only it needed. You do not have manualy activate/disable it.
You can configure flakes quantity, flakes speed.

Download, Activate, Be suprised!

== Installation ==

1. Download Archive and Upload it to your Wordpress;
1. Activate plugin;
1. Goto Options - Snow Configuration and configure plugin for your taste. It have some already configured presets.
